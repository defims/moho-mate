# moho-mate

> Moho 动画软件自动化助手

## 安装

```bash
curl -fsSL https://moho-mate.maohou.com/get | bash
```

## 使用

```bash
moho-mate --help
```

## 功能特性

- **IPC 模式** - 通过 Lua 脚本控制 Moho
- **渲染** - 批量渲染项目为 PNG/MP4/GIF/APNG
- **包管理器** - 安装和管理脚本包
- **自升级** - 内置升级机制
